DDR Daily Challenge Website:
by: Kenton Knelle

------------------------------------------------------------------------------------------------------------------------------------------------------------

Purpose of Website:

I wanted to create an easy-to-use site to act as a simplified Stepchart suggestion for a DDR Player, as well as allow the user to update and change their score.

This website utilizes an ASP.NET Core MVC workflow. The web-application’s use would be to randomize a DDR (Dance Dance Revolution) song for a user to play and insert their score. My project’s main components are a home page, a Daily Stepchart page, and a Chart Library link which is hyperlinked to a playlist containing the possible stepcharts that can appear.

------------------------------------------------------------------------------------------------------------------------------------------------------------

Features:


- Home Page contains instructions for maneuvering the site's tabs and for score input.

- Daily Stepchart Page allows the user to update and change their score with ambiance audio when the randomized song is visible.

- Chart Library link has a hyperlink to a playlist containing the possible stepcharts that can appear to study them.

- Stylized CSS and HTML features to aid baseline website functionality.

------------------------------------------------------------------------------------------------------------------------------------------------------------

Usage Instructions:

- Clone the Git repository to an easy-to-find folder on your device.

- Open the JetBrains Rider Solution file and press "Run".

- Follow the instructions on the home page.
